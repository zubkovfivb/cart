Implementation of todo list, with all its possibilities and limitation in 20 records; saved records and their state of the list stored in the local storage and cookies; upon initialization, data is taken from the local store, but if they are out there, then use the cookie; this application is made in two versions, using js and jquery.

Control:
		Mark all as complete - select all records 
		Clear X completed items - delete all selected records 
	text field:
		enter - add record 
	note: 
		check box - mark
		cross - delete 
		double click - edit 
		enter - save
